# 🎮 PCL整合包EXE生成器

## 📋 项目简介

这是一个专为新手设计的**PCL整合包安装器生成工具**，让你只需简单几步就能轻松创建属于自己的整合包安装程序！

### ✨ 核心功能

- **一键生成**：简单操作，快速生成可独立运行的EXE文件
- **无需编程**：友好的图形界面，零编程基础也能使用
- **自动打包**：整合包自动嵌入EXE，无需额外文件
- **即生成即用**：生成的EXE可直接分发给其他玩家
- **智能安装**：自动检测并安装整合包到PCL启动器

## 🛠️ 系统要求

| 运行模式 | 操作系统 | Python环境 | 硬件要求 |
|---------|---------|-----------|---------|
| **预编译生成器** | Windows 7/8/10/11或更高 | ❌ 无需 | 至少2GB内存，500MB可用磁盘空间 |
| **源码运行** | Windows 7/8/10/11或更高 | ✅ 需要Python 3.14+ | 至少2GB内存，500MB可用磁盘空间 |
| **生成的安装器** | Windows 7/8/10/11或更高 | ❌ 无需 | 至少2GB内存，500MB可用磁盘空间 |

## 📦 安装说明

### 方法1：预编译生成器（推荐 - 无需Python环境）
1. 下载项目压缩包并解压
2. 直接双击运行根目录下的 `PCL_Installer_Generator.exe` 文件
3. 等待程序启动，开始使用！

### 方法2：手动启动（需要Python环境）
1. 打开命令提示符
2. 进入项目目录
3. 输入 `python src/main.py` 并回车

### 二、制作EXE安装器

#### 准备工作
1. 确保已成功导出PCL整合包文件（.mrpack格式）
2. 确保已下载并解压本项目压缩包
3. 确保你的电脑满足系统要求

#### 步骤1：启动EXE生成器
1. 找到解压后的项目文件夹
2. 直接双击运行 `PCL_Installer_Generator.exe`（无需Python环境）
3. 等待程序启动，你将看到一个友好的图形界面
4. 界面上方显示 `🎮 PCL整合包EXE生成器 🎮` 标题

#### 步骤2：选择整合包文件
1. 找到界面中 `📦 1. 选择您的整合包文件` 区域
2. 点击右侧的 `浏览...` 按钮
3. 在弹出的文件选择窗口中，导航到你之前导出的整合包文件所在文件夹
4. 找到后缀为 `.mrpack` 的整合包文件并选中
5. 点击窗口右下角的 `打开` 按钮
6. 此时文件路径会显示在左侧的输入框中

#### 步骤3：设置输出EXE名称
1. 找到界面中 `🏷️ 2. 设置输出EXE名称` 区域
2. 在 `文件名:` 输入框中，为你的安装器取一个好听的名字
   - 示例：`我的模组包安装器`、`MC天堂整合包`
3. **注意事项**：
   - 不要包含特殊字符（如：`!@#$%^&*()`）
   - 名称不要过长（建议不超过20个字符）
   - 尽量使用中文或英文名称

#### 步骤4：生成EXE文件
1. 确认前面步骤已完成：
   - ✅ 已选择整合包文件
   - ✅ 已设置输出名称
2. 找到界面下方的 `🚀 3. 生成EXE文件` 按钮
3. 点击该按钮开始生成
4. 界面底部状态会依次显示：
   - `正在准备整合包文件...`
   - `正在生成EXE文件...`
5. 生成过程需要1-2分钟，请耐心等待
6. 生成成功后，会弹出提示窗口：`EXE文件生成成功！输出路径: dist/xxx.exe`
7. 点击 `确定` 关闭提示窗口

#### 步骤5：获取生成的EXE文件
1. 返回项目文件夹
2. 找到名为 `dist` 的文件夹并打开
3. 你将看到生成的 `.exe` 文件，这就是你的专属整合包安装器
4. 你可以：
   - 直接双击运行测试效果
   - 复制到桌面方便后续使用
   - 通过QQ、微信等方式分享给其他玩家

## 🎯 玩家使用方法（分享给他人）

当其他玩家收到你生成的EXE文件后，可按以下步骤使用：

1. 双击运行分享的EXE文件
2. 弹出安装器界面后，点击 `浏览...` 按钮
3. 选择PCL启动器所在目录
4. 确认目录选择正确后，点击 `一键安装并启动PCL` 按钮
5. 程序会自动执行：
   - 检查目录是否正确
   - 关闭正在运行的PCL进程
   - 将整合包文件复制到正确位置
   - 启动PCL启动器
6. 安装完成后，PCL会自动启动并开始导入整合包

## ❓ 常见问题

### Q1：双击 `PCL_Installer_Generator.exe` 没有反应怎么办？
**A1**：
- 检查系统是否满足最低要求
- 尝试以管理员身份运行
- 关闭杀毒软件重试
- 尝试使用源码运行方式

### Q2：生成EXE时出现错误怎么办？
**A2**：请检查：
- 整合包文件是否完整，无损坏
- 输出文件名是否包含特殊字符
- 是否有足够的磁盘空间（至少500MB）
- 整合包格式是否为 `.mrpack`

### Q3：生成的EXE文件很大，正常吗？
**A3**：正常！EXE文件包含：
- 完整的Python运行环境
- 你的整合包文件
- 安装器程序

通常大小在10-20MB之间，若整合包本身较大，EXE体积也会相应增加。

### Q4：生成的EXE可以在其他电脑上运行吗？
**A4**：当然可以！生成的EXE是完全独立的，无需在其他电脑上安装任何依赖。

### Q5：如何更新我的整合包？
**A5**：
1. 在PCL中重新导出更新后的整合包
2. 使用本工具重新生成一次EXE文件
3. 用新生成的EXE替换旧版本即可

### Q6：支持哪些整合包格式？
**A6**：目前仅支持 `.mrpack` 格式，这是PCL官方推荐的标准格式。

### Q7：生成的EXE会报毒吗？
**A7**：
- 本工具生成的EXE是安全的，不包含病毒
- 由于是新生成的程序，部分杀毒软件可能会误报
- 建议将生成的EXE添加到杀毒软件白名单中
