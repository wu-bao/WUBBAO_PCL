# 🎮 PCL整合包EXE生成器

## 📋 项目简介

这是一个专为新手设计的**PCL整合包安装器生成工具**，让你只需三步就能轻松创建属于自己的整合包安装程序！

### ✨ 核心功能

- **一键生成**：简单三步，生成可独立运行的EXE文件
- **无需编程**：友好的图形界面，无需任何编程知识
- **自动打包**：整合包自动嵌入EXE，无需额外文件
- **即生成即用**：生成的EXE可以直接分发给其他玩家
- **智能安装**：自动检测并安装整合包到PCL启动器

## 🛠️ 系统要求

- **操作系统**：Windows 7/8/10/11
- **Python版本**：Python 3.14（已自动安装）
- **硬件要求**：至少2GB内存，500MB可用磁盘空间

## 📦 安装说明

### 方法1：直接使用（推荐）

1. 下载项目压缩包并解压
2. 双击运行 `启动生成器.bat` 文件
3. 等待程序启动，开始使用！

### 方法2：手动启动

1. 打开命令提示符
2. 进入项目目录
3. 输入 `python -m src.main` 并回车

## 🚀 使用教程

### 第1步：选择整合包文件

1. 点击 `浏览...` 按钮
2. 在文件选择窗口中找到你的整合包文件（后缀为 `.mrpack`）
3. 选中文件并点击 `打开`

### 第2步：设置输出名称

- 在 `文件名:` 输入框中，为你的安装器取一个好听的名字
- 例如：`我的模组包安装器`
- 注意：不要包含特殊字符

### 第3步：生成EXE文件

- 点击 `🚀 3. 生成EXE文件` 按钮
- 等待程序生成（通常需要1-2分钟）
- 生成成功后，会弹出提示窗口

### 第4步：使用生成的安装器

1. 在项目目录下找到 `dist` 文件夹
2. 里面的 `.exe` 文件就是你的专属安装器
3. 把这个EXE文件分发给其他玩家吧！

## 🎯 玩家使用方法

当其他玩家收到你生成的EXE文件后：

1. 双击运行EXE文件
2. 点击 `浏览...` 选择PCL启动器所在目录
3. 点击 `一键安装并启动PCL` 按钮
4. 等待安装完成，PCL会自动启动！

## ❓ 常见问题

### Q1：生成EXE时出现错误怎么办？

**A1**：请检查：
- 你的整合包文件是否完整
- 输出文件名是否包含特殊字符
- 确保你有足够的磁盘空间

### Q2：生成的EXE文件很大，正常吗？

**A2**：正常！因为EXE文件包含了完整的Python环境和整合包，通常在10-20MB之间。

### Q3：生成的EXE可以在其他电脑上运行吗？

**A3**：当然可以！生成的EXE是独立的，无需安装任何依赖。

### Q4：如何更新我的整合包？

**A4**：重新生成一次EXE文件即可，替换掉旧的版本。

### Q5：支持哪些整合包格式？

**A5**：目前支持 `.mrpack` 格式的整合包，这是PCL支持的标准格式。
